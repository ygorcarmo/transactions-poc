# Rinha de Backend

## Ygor Carmo

To create the image run:

```
    docker build --tag "rinha" .
```


Submissão feita com:

- `nginx` como load balancer
- `postgres` como banco de dados
- `go` para api
- [repositório da api](https://github.com/ygorcarmo/rinha-de-backend-2024-q1)
